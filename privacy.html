Privacy Policy

Last updated: April 2026

Summary: LocalVoiceType does not collect, store, or transmit any personal data. All processing happens exclusively on your Mac.

1. Data Controller

Thomas Pallus
Contact: GitHub Issues

2. What data is processed?

LocalVoiceType processes only the following data:

  • Microphone audio: Voice recordings are processed exclusively on your device. They are never stored or transmitted to external servers. Audio data is discarded immediately after transcription.
  • AI model download: On first launch, the app downloads the Whisper model from HuggingFace (~632 MB). HuggingFace's privacy policy applies to this one-time download. After downloading, the model is stored locally — no further network access occurs during normal use.

3. No data collection

The app does not collect, store, or transmit:

  • Personal data
  • Usage statistics or telemetry
  • Crash reports to external services
  • Device information

4. Microphone permission

The app requires microphone access solely for speech transcription. You can revoke this permission at any time in System Settings → Privacy & Security → Microphone.

5. Accessibility permission

The Auto-Type feature (inserting transcribed text into other apps) requires Accessibility access. This permission is used exclusively to send keystrokes to the currently active application.

6. No third parties

The app does not use any analytics tools, tracking frameworks, advertising services, or third-party SDKs of any kind.

7. Changes

Material changes to this privacy policy will be published on this page. The date of the last update is shown at the top.

8. Contact

For privacy-related questions: GitHub Issues

Datenschutzerklärung

Zuletzt aktualisiert: April 2026

Kurzfassung: LocalVoiceType erfasst, speichert oder überträgt keine persönlichen Daten. Alle Verarbeitung findet ausschließlich lokal auf deinem Mac statt.

1. Verantwortliche Person

tomekcp
Kontakt: GitHub Issues

2. Welche Daten werden verarbeitet?

LocalVoiceType verarbeitet ausschließlich die folgenden Daten:

  • Mikrofon-Audio: Die Sprachaufnahme wird ausschließlich lokal auf deinem Gerät verarbeitet. Sie wird weder gespeichert noch an externe Server übertragen. Nach der Transkription werden die Audiodaten sofort verworfen.
  • KI-Modell-Download: Beim ersten Start lädt die App das Whisper-Modell von HuggingFace (~632 MB). Dabei gelten die Datenschutzbestimmungen von HuggingFace. Nach dem einmaligen Download wird das Modell lokal gespeichert — kein weiterer Netzwerkzugriff im normalen Betrieb.

3. Keine Datenerhebung

Die App erhebt, speichert oder überträgt keine:

  • Personenbezogenen Daten
  • Nutzungsstatistiken oder Telemetrie
  • Absturzberichte an externe Dienste
  • Geräteinformationen

4. Mikrofon-Berechtigung

Die App benötigt Zugriff auf das Mikrofon ausschließlich für die Sprachtranskription. Du kannst die Berechtigung jederzeit unter Systemeinstellungen → Datenschutz & Sicherheit → Mikrofon widerrufen.

5. Barrierefreiheits-Berechtigung (Accessibility)

Für die Auto-Type-Funktion (Text in andere Apps einfügen) benötigt die App Accessibility-Zugriff. Diese Berechtigung wird ausschließlich zum Senden von Tastatureingaben an die aktive App verwendet.

6. Keine Drittanbieter

Die App verwendet keine Analyse-Tools, Tracking-Frameworks, Werbedienste oder sonstigen Drittanbieter-SDKs.

7. Änderungen

Wesentliche Änderungen dieser Datenschutzerklärung werden auf dieser Seite veröffentlicht. Das Datum der letzten Aktualisierung steht oben.

8. Kontakt
